Key Features

Keep tally of occupancy levels

It may be a simple counter, but its an intuitive way to tell where current occupancy levels are at, and if you’re within requirements set by your local health organization.

Multi-client Support

Have more than one entrance or exit? The app supports multiple simultaneous connections to accommodate multiple entry points.

Set up a space, share your unique URL with the people to be stationed at both entrances, and start keeping track.

Works on any Smart Device

Made for phones and tablets, but will work on any semi-modern browser. Internet access is required since we handle all the multi-client support on the backend, but that’s about the only real requirement.

Private

We don’t store any personal data, or keep data long term. Our system just keeps track of a number assigned to a space, but that’s all it knows.

We do use analytic data to keep an eye on web traffic , but have anonymization features in place to mask IP addresses.
